# Mac Sync Notes

## Copy The Codex Pet

On the Mac, copy this folder:

```text
codex-pet-fujie/
```

to:

```text
~/.codex/pets/fujie/
```

The final Mac path should contain:

```text
~/.codex/pets/fujie/pet.json
~/.codex/pets/fujie/spritesheet.webp
```

Restart Codex after copying the folder if the pet list does not refresh.

## Rebuild A Desktop Shell On Mac

Use `assets/high-res-rows/` as the visual source for any Mac-native desktop pet
wrapper. The Codex atlas works for the Codex app itself, but the high-resolution
row strips are sharper for a floating desktop window.

The behavior spec is in:

```text
design/FUJIE_DESIGN_SPEC.md
```
